## リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| Claude API値上げ | 粗利率低下 | Batch API/キャッシュで現状コストの50%以下に最適化可能。Haiku併用も検討 |
| バイラル係数Kが低い | 成長停滞 | SEOコンテンツ投資でオーガニック獲得を補完 |
| 解約率が想定以上 | MRR伸び悩み | NPS改善、週次レポートの価値向上、カスタマーサクセス |
| Free乱用 | API赤字拡大 | IP+Cookie制限で実質月5回。Turnstile追加 |
| 競合参入 | シェア喪失 | プロンプト品質の継続改善（模倣困難な深層価値） |

## 備考

- 本計画はコスト実績値（¥17/分析）に基づく保守的な見積もり
- Batch API適用で変動費は最大50%削減可能（M6以降に検討）
- プロンプトキャッシュ適用で同一セッション内は最大90%削減可能
- 人件費・マーケティング広告費は本P&Lに含まない（変動費+インフラのみ）
- 為替変動リスク：$1=¥130-170のレンジで粗利率82-89%
