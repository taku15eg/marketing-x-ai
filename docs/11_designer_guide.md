# Publish Gate デザイナー向け実装指示書

**版：v3.0　更新日：2026-03-08**

---

## v3.0 変更サマリー

- Webダッシュボード6タブが主要デザイン対象
- Chrome拡張Side Panelは簡略版（リテンション/リマーケ用）
- Ahrefs Site Explorer型UI（URL入力 → 結果一覧 → 深掘り）
- 信頼のブルー（#1B3A5C系）基調
- v8-enrichedのデザイン言語踏襲
- 共有URL閲覧ページのOGPデザイン
- ロックタブのデザイン（他タブはロックアイコン + プラン名表示）
- 浦田型が圧倒されない設計

## 1. 設計方針

### コアコンセプト

**Ahrefs Site Explorer型：URL入力 → 結果一覧 → 深掘り**

チャットUIではない。1つのURLを入れるだけで全体像が見え、タブで深掘りする。「何を聞くべきかすら知らない」浦田型が、質問不要で答えにたどり着ける設計。

### 3つの設計原則

1. **URL入力だけで始まる** — 追加入力を要求しない。入力フォームは1つだけ
2. **情報の階層化** — 概要→詳細→依頼書と段階的に深掘り。浦田型が圧倒されない
3. **共有が自然に起きる** — 分析結果の共有URLが目立ちすぎず、でも見逃されない位置

### デザイン言語

v8-enrichedデモで確立したトーン（専門性 + 信頼性 + 実用性）を踏襲する。
「分析ツール」のプロフェッショナル感を保ちつつ、初見の浦田型でも「何をすればいいか」が一目でわかる明快さ。

## 3. カラーシステム

### メインカラー

```css
:root {
  /* プライマリ：信頼のブルー */
  --primary: #1B3A5C;
  --primary-light: #EDF2F7;

  /* アクセント：アクションブルー */
  --accent: #2563EB;

  /* セマンティック */
  --success: #059669;
  --warning: #D97706;
  --danger: #DC2626;

  /* テキスト */
  --text: #1E293B;
  --text2: #64748B;

  /* サーフェス */
  --bg: #F8FAFC;
  --surface: #FFFFFF;
  --border: #E2E8F0;

  /* ロックタブ */
  --lock: #94A3B8;
}
```

### セマンティックカラーの用途

| 用途 | カラー | 使用箇所 |
|---|---|---|
| ヘッダー・ロゴ・主要テキスト | `--primary` #1B3A5C | 信頼感の演出 |
| CTAボタン・リンク・アクティブタブ | `--accent` #2563EB | 行動を促す |
| 正常・良好・改善済み | `--success` #059669 | 診断結果、完了状態 |
| 注意・要確認 | `--warning` #D97706 | 中程度の課題 |
| エラー・重大課題・薬機法違反 | `--danger` #DC2626 | 緊急性の高い項目 |
| ロックされたタブ・無効要素 | `--lock` #94A3B8 | 上位プラン誘導 |

### ダークモード

Phase 0.5-1ではライトモードのみ。Phase 2以降でダークモード対応を検討。

## 8. コンポーネント仕様

### 8.1 ボタン

| バリアント | 背景 | テキスト | 用途 |
|---|---|---|---|
| Primary | `--accent` | 白 | 主CTA（分析する、共有する） |
| Secondary | 透明 | `--accent` | 副CTA（依頼書DL、計測設計） |
