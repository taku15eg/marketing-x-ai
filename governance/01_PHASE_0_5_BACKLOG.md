## ステータス定義

| ステータス | 意味 | 遷移条件 |
|-----------|------|---------|
| `NOT_STARTED` | 未着手 | — |
| `IN_PROGRESS` | 実装中 | Claude Code が着手した時点 |
| `IMPLEMENTED_UNVERIFIED` | 実装済み・実動作未確認 | コードが存在しテストは通るが、実環境での動作確認が未了 |
| `VALIDATED_DONE` | 実動作確認済み・完了 | 実環境（またはそれに準ずる環境）で動作確認が完了 |

> 重要: `VALIDATED_DONE` は実動作確認後のみ使用する。テスト通過のみでは `IMPLEMENTED_UNVERIFIED` に留める。
