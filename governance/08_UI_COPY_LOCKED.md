# 08_UI_COPY_LOCKED.md — UIコピー管理台帳

> ## 暫定ロックの定義
>
> このファイルに記載されたUIコピーは **「暫定ロック」** 状態である。
>
> - **暫定ロックとは**: 現在の実装コードから抽出した文言であり、人間による正式承認レビューは未了。
>   ただし、Claude Code による独断変更は禁止する。
> - **正式承認済みではない**: 人間がレビューし「正式ロック」と判定するまで、内容の修正可能性がある。
> - **Claude Code の義務**: このリストに載っている文言を変更する必要がある場合、
>   `02_DECISION_LOG.md` に記録し、人間にエスカレーションすること。独断変更は禁止。
> - **正式承認の手続き**: `02_DECISION_LOG.md` の DEC-003 を参照。UIコピー正式承認レビューが OPEN として登録されている。
>
> ステータス遷移: `暫定ロック` → 人間レビュー → `正式ロック`

---

## 変更ルール

1. このファイルに記載されたコピーは**暫定ロック**。Claude Code が独断で変更しない
2. 新規UIコピーが必要な場合、人間に提案し、承認後にこのファイルに追記する
3. コピーの修正が必要な場合、`02_DECISION_LOG.md` に記録し、人間にエスカレーションする
