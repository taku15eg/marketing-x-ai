## ディレクトリ構造と責務

| ディレクトリ | 責務 | 変更範囲 |
|------------|------|---------|
| `src/dashboard/app/` | ページルーティング・レイアウト | 既存ページの修正のみ。新規ページは人間承認 |
| `src/dashboard/components/` | UIコンポーネント | 既存コンポーネントの修正のみ。新規は人間承認 |
| `src/dashboard/lib/` | ビジネスロジック | バグ修正・リファクタリング。外部仕様変更は人間承認 |
| `src/dashboard/__tests__/` | テスト | 自由に追加・修正可能 |
| `src/extension/` | Chrome拡張 | 既存機能の修正のみ。新機能は人間承認 |
| `src/proxy/` | APIプロキシ | バグ修正のみ。エンドポイント追加は人間承認 |
| `docs/` | 仕様書 | 変更禁止。人間のみが更新 |
| `governance/` | 運用ガバナンス | ステータス更新・ログ追記のみ。構造変更は人間承認 |

## 変更してよいもの（人間承認不要）

- テストの追加・修正
- バグ修正（外部仕様を変えない範囲）
- 型定義の修正
- lint / format の修正
- コード内コメントの追加
- リファクタリング（外部仕様を変えない範囲）
- `governance/02_DECISION_LOG.md` への記録追加
- `governance/01_PHASE_0_5_BACKLOG.md` のステータス更新（DONE化）

## 変更に人間承認が必要なもの

- 新規ページ・コンポーネント・APIエンドポイントの追加
- npm パッケージの追加
- データベーステーブル・カラムの追加
- 外部サービスとの連携追加
- UIコピーの変更
- セキュリティ関連の仕様変更
- 分析プロンプトの構造的変更
- 仕様書（`docs/`）の更新

## 技術スタック（確定）

| 領域 | 技術 | 変更 |
|------|------|------|
| Web | Next.js + Tailwind CSS | 禁止 |
| Chrome拡張 | Manifest V3 + Side Panel API | 禁止 |
| バックエンド | Cloudflare Workers | 禁止 |
| DB | Supabase (PostgreSQL + Auth + RLS) | 禁止 |
| AI | Claude API + Vision API | 禁止 |
| 決済 | Stripe | 禁止 |
| デザイン | Noto Sans JP / #1B3A5C系 | 禁止 |

> 技術スタックの変更は人間のみが判断する。
