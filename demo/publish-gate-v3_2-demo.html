The Problem
マーケターが毎日ぶつかる4つの壁

3名のインタビューから抽出した、立場を超えて共通する痛み。

LAYER 01

なぜ数字が悪いかわからない

データはある。でも「なんでこの数字なのか」が見えない。分析だけに時間が消える。

「数字が悪いのはわかっているが、なんでその数字になっているのかがわからない」
LAYER 02

何をすればいいかわからない

課題は感じている。でも確からしい仮説が立てられず、施策の優先度がつけられない。

「確からしい課題仮説とかイシューの特定ができてない状態で迷う」
LAYER 03

実装に渡すと壊れる

指示書に2時間。渡しても意図が伝わらず手戻り。分業の摩擦でスピードが出ない。

「マーケのこと詳しくない開発者に細かく指示書を書くのが面倒」
LAYER 04

やったことが残らない

施策を回す時間より、ログや報告書を作る時間のほうが長い。学びが属人化する。

「施策を回してる時間より、ログを残す・報告書を作る時間の方が多い」
Features
4つの壁を、3つの機能で貫通する
